# Proyecto Final Front III
Este proyecto se llevó a cabo como parte de la evaluación final de la materia FrontEnd III en 
el marco del programa Certified Tech Developer de Digital House. Fue desarrollado utilizando 
React con el propósito de exponer los conocimientos adquiridos a lo largo de todo el curso.

## Requisitos
Es indispensable clonar este proyecto en un repositorio privado, para poder trabajar de forma 
individual o en dupla.
Para nuestro caso en dupla: https://github.com/sandyFit/Final-Front-III.git
Seguimos los pasos correspondientes en la terminal para hacer merge
con cada una de las ramas creadas desde local al proyecto en línea. De esta 
forma integrar los trabajos compartidos.
